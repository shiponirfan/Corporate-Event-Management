###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: `"12"`

<i>The sum function takes two parameters, a and b, and attempts to add them together using the + operator. When sum(1, "2") is called, JavaScript will perform type coercion and convert the number 1 to a string and concatenate it with the string "2", resulting in the string "12". So, the function returns the string "12".</i>

</p>
</details>

###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
const food = ["🍕", "🍫", "🥑", "🍔"];
const info = { favoriteFood: food[0] };

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `['🍕', '🍫', '🥑', '🍔']`

<i>In the provided code, the info.favoriteFood property is initially assigned the value of food[0], which is "🍕". Later, it is reassigned the value "🍝". However, this reassignment does not affect the food array. So, when logging the food array, it remains unchanged, and the output is ['🍕', '🍫', '🥑', '🍔'].</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: `Hi there, undefined`

<i>In the provided code, the sayHi function expects a name parameter, but when sayHi() is called without passing any arguments, the name parameter is undefined. Therefore, the function returns "Hi there, undefined," where "undefined" is the value of the name parameter that was not provided during the function call.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>In the provided code, the forEach method iterates over the nums array and increments the count variable by 1 for each element in the array that is truthy. Since all elements except 0 in the nums array are truthy, the count variable is incremented for each of them, resulting in a final value of 3. Therefore, the output of the count is 3.</i>

</p>
